Execute run.sh or run.py to start transcriptions. 
At first executions all needed libraries will be created and also all needed folders.
Videos to transcribe must be in .mp4 format and they have to be placed in 'source' folder 
inside this directory. If doesn't exist execute run.py, place videos in created folder and execute run.py again.

Set up transcripion language in config.py. Edit it with a text editor
